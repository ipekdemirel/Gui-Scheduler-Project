# CS 305 – Operating Systems  
## Process Scheduling Simulator

This project is a **CPU Process Scheduling Simulator** developed for the **CS 305 Operating Systems** course.  
It simulates different scheduling algorithms and allows users to run them through both **CLI** and **GUI** interfaces.

---

# How to Run

## Run with CLI

1. Open a terminal in the project directory.
2. Run the following command:

```bash
py scheduler.py processes.txt

Note:
Round Robin time quantum is set to 3 in the source code (scheduler.py).


Executable Version (.exe)

The project also includes a standalone Windows executable created using PyInstaller.

How to Run (.exe):
1. Open the folder
2. Run:
   gui_scheduler.exe

No Python installation is required to run the executable.
