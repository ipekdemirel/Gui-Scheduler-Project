CS 305 – Operating Systems
Process Scheduling Simulator

How to Run (CLI):
1. Open terminal in the project directory
2. Run:
   py scheduler.py processes.txt

How to Run (GUI):
1. Open terminal in the project directory
2. Run:
   py gui_scheduler.py
3. Select processes.txt and click "Run All"

Requirements:
- Python 3.10+
- No external libraries required
Round Robin Time Quantum:
- Default value is 3
- Can be changed from the GUI before running

Note:
Round Robin time quantum is set to 3 in the source code (scheduler.py).

------------------------------------
Executable Version (.exe)

The project also includes a standalone Windows executable created using PyInstaller.

How to Run (.exe):
1. Open the "dist" folder
2. Run:
   gui_scheduler.exe

No Python installation is required to run the executable.
